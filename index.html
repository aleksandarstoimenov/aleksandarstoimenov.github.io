Maquette PLE

Interactive et explorable

Navigation

• Utiliser la souris pour tourner autour de la maquette.

• Clickez sur une pièce pour zoomer.

• Sélectionner un objet

Information

• Une fois l'objet cliqué, des informations s'affichent.

Guide d'entretien du logement

- Clique une fois sur une pièce pour zoomer dessus.

- Clique encore une fois sur un élément pour afficher les informations d'entretien.

- Clique deux fois pour dézoomer.
